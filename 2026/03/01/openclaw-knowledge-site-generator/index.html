第五步：绑定自定义域名（1 分钟）

Vercel 部署完成后，你可以绑定自己的自定义域名，让网站更专业。

操作步骤：

  1. 访问 Vercel 项目设置

    • 打开 Vercel Dashboard
    • 选择你的项目（openclaw-learning）
    • 点击 “Settings” → “Domains”

  2. 添加域名

    • 点击 “Add” 按钮
    • 输入你的域名，如 openclaw.studyai.ltd
    • 点击 “Add” 确认

  3. 配置 DNS 记录

    • Vercel 会显示需要添加的 DNS 记录
    • 登录你的域名 DNS 管理后台（如 Cloudflare、阿里云 DNS）
    • 添加以下记录：
      • A 记录：主机记录 @，记录值 76.76.21.21
      • CNAME 记录：主机记录 www，记录值 cname.vercel-dns.com

  4. 等待 DNS 生效

    • DNS 记录通常需要几分钟到几小时生效
    • Vercel 会自动检测 DNS 配置状态
    • 生效后，Vercel 会自动配置 HTTPS 证书

  5. 验证配置

    • 在浏览器访问你的域名：https://openclaw.studyai.ltd
    • 检查 HTTPS 证书是否正常
    • 验证网站内容是否正确显示

示例配置（Cloudflare）：

类型 名称 内容 代理状态
A @ 76.76.21.21 DNS Only
CNAME www cname.vercel-dns.com DNS Only

注意事项：

  • ⚠️ DNS 生效时间因服务商而异，通常 5-30 分钟，最长可能 48 小时
  • ⚠️ Vercel 会自动配置 HTTPS，无需手动申请证书
  • ⚠️ 如果使用 Cloudflare，确保 CDN 代理状态为 “DNS Only”（灰色云朵）
  • ⚠️ 绑定后，原 Vercel 域名仍然可用，会自动重定向到自定义域名
