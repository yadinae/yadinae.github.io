Hermes 的记忆宝库 —— 让 AI 真正记住你

解决的问题：每次跟 Hermes 聊天都要重新自我介绍吗？它能记住我之前说过的话和偏好设置吗？

记忆，是 Hermes 最与众不同的能力

用过 ChatGPT 或 Claude 的人都知道，每次开一个新对话，AI 就”失忆”了——你得重新说一遍背景、重新设定风格。

Hermes 不一样。它有一个”记忆宝库”，你告诉它的每一件重要的事，它都会记下来，下次聊天时自动想起来。

实际体验一下

第一天

你：你好，我是个技术博主，写 AI 相关的文章。

Hermes：收到！我会记住你是技术博主，专注 AI 领域。

过了几天

你：我又来了，上次说的那篇关于 Agent 的文章写完了，准备发新的。

Hermes：欢迎回来！我记得你是 AI 领域的博主。上次你在写 Agent 的文章，现在写完了？新文章是关于什么主题的？

看到了吗？它记得你是谁、你做什么的、上次聊了什么。不需要重新介绍。

又过了一周

你：帮我看看这篇文章有没有语病。

Hermes：好的，我注意到你平时喜欢用简洁直白的风格，我会按这个标准来审阅。

因为它通过几次交流已经掌握了你的写作风格偏好。

记忆是怎么工作的？

Hermes 的记忆系统分为三个层次：

📝 会话记忆

每次对话中，Hermes 会记住你刚刚说过的话。关掉终端再打开，这个层次的记忆就清空了——就像人类短暂记忆一样。

📚 持久记忆

这是 Hermes 的”长期硬盘”。当你告诉它重要信息时——比如你的职业、偏好、正在做的项目——它会写入 MEMORY.md 文件。下次启动时自动加载。

你：记住我每天早上需要看 GitHub 动态。

Hermes：已记录！以后每天早上我会先问你 GitHub 动态。

🧠 习惯学习

Hermes 还有一个更厉害的能力——它会观察你的使用模式。

如果你连续三天都问它”今天有什么 AI 新闻”，到第四天它可能会主动说：

“早上好！需要我像前几天一样，帮你汇总今天的 AI 新闻吗？”

这不是你”教”它的，是它自己发现的。

查看和管理记忆

你也可以随时查看 Hermes 记住了什么：

你：你现在都记得关于我的什么？

Hermes：目前关于你的记录有：

  • 你是 AI 领域的技术博主
  • 你喜欢简洁直接的沟通风格
  • 你每天早上关注 GitHub 动态
  • 你正在使用 Stellar 主题管理博客

需要我修改或删除任何记录吗？

想删除某条记忆？直接告诉它：

你：把我关于 GitHub 动态的那条记录删掉。

就这么简单。

记忆的妙用

使用场景 你说的话 效果
设置风格 “记住我喜欢简短回答” 以后回答自动精简
记录偏好 “以后推荐工具优先推荐开源的” 推荐策略永久调整
追踪项目 “我在做一个叫 Hermes 的知识库” 后续帮你完善内容
重要日期 “记住下周三是我交稿截止日” 到时提醒你

记忆 vs 技能

你可能想问：记忆和技能有什么区别？

简单来说：

  • 记忆 = Hermes 知道关于你的事（被动信息）
  • 技能 = Hermes 会帮你做的事（主动能力）

下一篇文章我们会详细介绍技能系统。

隐私安全

记忆文件就保存在你的电脑上。Hermes 不会把你的记忆上传到任何云端（除非你主动配置了远程备份）。所有数据你随时可以查看、修改、删除——就是一些文本文件而已。

你完全掌控自己的数据，这是开源自托管最大的好处。
