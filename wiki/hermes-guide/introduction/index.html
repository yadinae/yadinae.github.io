Hermes入门指南｜初识 Hermes

发表于2026-05-20|更新于2026-05-28|Hermes

Hermes 是什么？—— 一个会自我进化的 AI 队友

解决的问题：听说 Hermes 最近很火，它到底是什么？跟我用的 ChatGPT、Claude 有什么不一样？我为什么要用它？

一句话说清楚

Hermes Agent 是由 Nous Research 开发的开源 AI 助手。它跟其他 AI 最大的区别在于：

别的 AI 跟你说完话就忘了，Hermes 会用记忆记住你，还会自己学新本事，越用越聪明。

想象一下：你招了一个实习生，第一天他啥都不会，但每次教完他都会记住，而且自己总结出工作方法。一个月后，他已经是你最得力的助手了。Hermes 就是这样一个”AI 实习生”。

先看一个真实场景

让对比更直观，看看你分别用 ChatGPT 和 Hermes 做同一件事会怎样：

你的请求 ChatGPT 的做法 Hermes 的做法
“帮我整理今天的 GitHub 通知” 告诉你步骤，但每次都要重新说 做一次就记住格式，下次直接出结果
“每天上午 8 点给我发日报” 做不到（它不会主动找你） 设置一次，以后自动发送
“帮我写个周报模板” 给你一个通用模板 记住你的项目风格，以后越写越贴切

Hermes 的五大核心能力

1. 🧠 持久记忆

Hermes 有自己的”大脑”——一个叫 MEMORY.md 的文件。你告诉它的每件事它都记在这里。下次聊的时候，它记得你之前说过什么、喜欢什么、讨厌什么。

举个例子：你跟它说”我主要在写 Python 后端”，下次问它代码问题，它会自动用 Python 来回答，不用你再重复。

2. 🛠️ 自建技能

这是 Hermes 最酷的能力。当它发现自己在重复做某件事，它会自己写一个”技能说明书”，下次直接套用。这些技能就是简单的 Markdown 文件，你也能看懂、修改、删除。

3. 📱 多渠道触达

Hermes 不只是一个终端聊天框。它可以通过 Telegram、Discord、Slack、邮件等 14 种方式跟你交流。

  • 在电脑前？用终端直接聊
  • 出门在外？掏出手机在 Telegram 上找它
  • 开会时？发个邮件它也能回复

4. 🔄 模型自由

Hermes 不绑定任何一家 AI 公司。你可以随时切换它的大脑：

目前支持 20+ 家模型供应商：

  • 最强性能：Claude、GPT-5.4
  • 超高性价比：DeepSeek、GLM
  • 完全免费：本地模型（用自己的电脑跑）
  • 中文首选：GLM（智谱）、Qwen（通义千问）

5. ⏰ 定时任务

你让它”每天早上 8 点把今天的 GitHub 通知摘要发到我的 Telegram”，它就会做到。不需要服务器定时任务、不需要脚本——Hermes 内置了这个能力。

谁在用 Hermes？

编程达人：用终端跟它配合写代码，比 Copilot 更主动
效率狂热者：设置自动化流程，让 Hermes 在你睡觉时干活
手机控：在 Telegram 上跟 Hermes 聊天，随时随地用 AI
