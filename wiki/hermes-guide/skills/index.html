Hermes入门指南｜技能系统

发表于2026-05-25|更新于2026-05-28|Hermes

教 Hermes 新本事 —— 技能系统入门

解决的问题：有些事我经常让 Hermes 做，能不能让它学会并记住怎么做，下次直接说”用老方法”就行？

什么是技能？

技能是 Hermes 最强大的功能之一。简单来说：

技能 = 你教 Hermes 做一件特定事情的方法，它学会后可以反复使用。

想象一下这些场景：

你：帮我把这篇技术文章转成小红书风格的文案。

Hermes：好的。我看你上周也做过类似的事情，需要我用上次那种转换方式吗？

你：对，就用那个方法。

这就是技能在起作用——Hermes 发现你做了两次类似的事，就自动总结出了一个”技能”，下次可以直接复用。

Hermes 怎么学会技能的？

🤖 自动学习（最神奇的部分）

你不需要专门教 Hermes 技能。它会自己观察你的使用模式。

举个真实的例子：

第1次：你让 Hermes 把一篇博客总结成三句话。

第2次：你又让它做同样的事。

第3次：Hermes 说：”我发现你经常让我做文章总结，我帮你总结了一个标准流程，以后直接说’老方法总结’就行。”

Hermes 会这样跟踪：

  1. 注意到你在重复做类似的事情
  2. 自动生成一个技能文件
  3. 下次主动提出复用

✋ 手动教它

你也可以直接教它新技能：

你：我教你一个技能。以后我说”日报”，你就按这个格式来：今天的完成事项、遇到的问题、明天的计划。每一项用列表。明白了吗？

Hermes：明白了！我已经记下了这个”日报”技能。以后你说”日报”，我就按这个格式生成。

技能长什么样？

每个技能其实就是一个文本文件，存在你的电脑上。你可以用任何文本编辑器打开查看和修改。

技能的大致结构是这样的：

你可以随时打开这些文件看看，修改或删除都可以。不需要任何编程知识。

实用的技能推荐

这里有一些其他用户常用的技能，也许能给你启发：

技能名称 作用
✍️ 文章润色 把你的草稿润色成发布级质量
📝 会议纪要 把对话整理成结构化会议记录
🐛 问题排查 一步步排查技术问题
📊 数据分析 对 CSV 数据做快速分析
🌐 翻译校对 中英互译并保持语气自然

技能 vs 记忆，再理一遍

记忆 技能
是什么 关于你的事实 做事的方法
例子 “你是技术博主” “用小红书风格写文案”
怎么来的 你告诉它或它观察到 自动总结或你教它
可以改吗 随时改 随时改
存哪里 MEMORY.md / USER.md skills/ 目录下的 .md 文件

管理你的技能

查看 Hermes 现在会什么技能：

你：你现在会什么技能？

Hermes：目前我掌握了这些技能：

  1. 📰 文章总结（自动学习）
  2. 📝 日报生成（你教我的）
  3. ✍️ 博客润色（你教我的）
  4. 🌐 中英翻译（内置）

你想试试哪个？或者想教我新的？

想删除某个技能？

你：把”日报生成”的技能删掉，我换一种格式。

小结

技能系统让 Hermes 从一个”通用 AI”变成了”你的专属 AI”。用得越多，它学会的技能就越多，越来越贴合你的工作方式。

核心思想：不要重复教它，教会一次，永久复用。
